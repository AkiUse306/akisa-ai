# Training Pipeline

This directory contains training pipeline design for AKISA-AI.

## Phase 3 focus

- Collect multi-modal examples for vision and text workflows.
- Build dataset pipelines for conversation, reasoning, and document understanding.
- Add training support for code generation and agent planning tasks.

## Example pipeline steps

1. Ingest raw data from documents, code repos, and conversations.
2. Clean and normalize text and metadata.
3. Create training splits for supervised and reinforcement learning.
4. Export datasets in JSONL or Parquet format for model fine-tuning.

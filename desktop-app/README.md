# Desktop App

This directory contains a lightweight desktop companion prototype for AKISA-AI.

## Included files

- `desktop-manifest.json` — manifest describing the desktop companion.
- `index.html` — static prototype UI linking to the AKISA workspace.
- `package.json` — helper script for serving the prototype locally.

## Goals

- Local AI companion shell
- File indexing and desktop automation
- Cross-platform packaging via Electron, Avalonia, or Tauri

## Run locally

```bash
cd desktop-app
npm install
npm start
```

# Mobile App

This directory contains a static mobile companion prototype for AKISA-AI.

## Included files

- `app-manifest.json` — mobile app manifest for future native clients.
- `index.html` — mobile shell preview that links to the workspace.
- `package.json` — helper script for local preview.

## Goals

- Voice assistant experience
- AI chat on the go
- Camera and OCR features
- Future React Native or .NET MAUI implementation

## Run locally

```bash
cd mobile-app
npm install
npm start
```
